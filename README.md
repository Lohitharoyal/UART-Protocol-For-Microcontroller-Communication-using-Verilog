# UART-Protocol-For-Microcontroller-Communication-using-Verilog

## Features
-Baud Rate Generation
- UART TX/RX Modules
- FIFO-Based Data Buffering
- Parity Error Detection
- Start/Stop Bit Handling
- Configurable Baud Rates
- Functional Verification using Testbench
## Modules
- baud_generator.v
- uart_tx.v
- uart_rx.v
- fifo.v
- parity_generator.v
- parity_checker.v
- uart_controller.v
- uart_top.v
- uart_tb.v
